Minecraft Command Tabs

Commands

Give Command:

/give @p minecraft:diamond 64

Teleport Command:

/tp @p 100 64 200

Summon Command:

/summon minecraft:creeper

Weather Command:

/weather clear

Time Set Command:

/time set day

Gamemode Command:

/gamemode survival @a

Kill Command:

/kill @e[type=!minecraft:player]

Effect Command:

/effect @a minecraft:speed 60 5

Fill Command:

/fill 0 64 0 10 70 10 minecraft:stone

Execute Command:

/execute as @a at @s run say Hello, Minecrafters!

Enchant Command:

/enchant @p minecraft:sharpness 5

Scoreboard Command:

/scoreboard objectives add kills playerKillCount

Tellraw Command:

/tellraw @a {"text":"Welcome to the server!","color":"yellow"}

Setblock Command:

/setblock 10 64 10 minecraft:redstone_block

Execute at Entities Command:

/execute as @e[type=skeleton] at @s run summon minecraft:zombie

Spread Players Command:

/spreadplayers 0 0 100 100 true @a

Give Custom Named Item Command:

/give @p minecraft:diamond_sword{display:{Name:'{"text":"Legendary Sword"}'}} 1

Execute if Block Command:

/execute if block 0 64 0 minecraft:diamond_block run say Found a diamond block!
